Retrain Models

Upload a labeled dataset, and CreditSight will retrain Decision Tree, Random Forest, Logistic Regression, and XGBoost on that data. The latest metrics will replace the previous saved models.

The file must include the rating column. If it is not named `Rating`, enter the target column name below.

Manual Company Assessment

Enter the company's raw financial statement values. CreditSight will calculate the 24 model features, run the selected model, and show the prediction with feature contributions.

Use the same units throughout, such as millions for cash flow and revenue. Required sectors are the same as Dataset A.

Batch CSV Upload

Upload a CSV or Excel file containing the model features. The batch flow validates the required columns, warns about unsupported extras, and returns a prediction for each row.

The file should contain the exact feature column names expected by the selected model.
